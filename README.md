# Dockerized Wordpress Enviroment

## Requirements
* `git`
* `make`
* `docker`

## How to use
1. Copy your wordpress file into `./src` folder
2. Run `Make up` for up and run docker project.

## DB Info
 * DB hostname : `wp-mysql`
 * DB name : `root`
 * DB password : `root`

 ## Access DB Service

 * phpMyAdmin will run on `8001` port.
